# 智能油烟机（Smart Hood System）— 硬件约束规划 v3.1

> 版本：v3.1 | 日期：2026-05-24 | 状态：**已审批 (PB11修复)**
> 硬件路线：**适配Mini H562 V1开发板** | ACS712：**5V供电+电阻分压** | 驱动：**现成电机模块(DRV8833/L298N)**

---

## ⚠ v3.1 变更说明

**PB11 未引出！** Mini H562 V1 的 LQFP-100 Pin48 (PB11) 没有接到任何排针。因此：
- **PMS7003 改用 USART3 (PD8/PD9)**, AF7复用, 外设不变
- **不再使用 USART2**

## 二、固件改动清单

基于Mini H562 V1引脚映射，**现有固件需要约10处改动**：

### 2.1 `firmware/Core/Inc/config.h` 改动

```c
// ====== 时钟配置 (Mini H562 使用 8MHz HSE) ======
// 原固件假设25MHz HSE，改为8MHz
// PLLM=2, PLLN=125, PLLR=2 → VCO=500MHz, SYSCLK=250MHz

// ====== ACS712 参数 (5V供电 + 1:2分压) ======
#define HALL_SENSITIVITY   0.0925f   // 原 0.185f → 分压后等效灵敏度减半
#define HALL_OFFSET_V      1.25f     // 原 1.65f → 5V供电零电流=2.5V, 分压后=1.25V
```

### 2.2 `firmware/Core/Src/main.c` 改动

```c
I2C_HandleTypeDef  hi2c1;  // I2C1: PB6/PB7
UART_HandleTypeDef  huart1; // USART1: PA9/PA10 → OpenMV
// USART3: PD8/PD9  → PMS7003 (仅引脚改，外设仍是huart3，固件不动)

// GPIO_Init()中:
// - PA2 Motor EN → 改为 PD7
// - TIM2_CH1→TIM1_CH1 (PA8)
// - 新增 PC5(Motor DIR), PE0(Buzzer)
// - 按键 PA3/PA4/PA5 → PA0/PA1/PA15 (板上按键)

// ADC1_Init()中:
sConfig.Channel = ADC_CHANNEL_4;  // PC4替代PA1

// TIM1_PWM_Init() 替代原 TIM2_PWM_Init()

// USART3 → PMS7003: 仅引脚从PB10/PB11改为PD8/PD9，外设huart3不动
```

### 2.3 `firmware/Drivers/Hall/hall.c` 改动

```c
float Hall_ADCToCurrent(uint16_t adc_value) {
    float voltage = ((float)adc_value / 4095.0f) * 3.3f;
    // 1:2分压后 ADC读数=实际电压/2
    float actual_voltage = voltage * 2.0f; // 还原ACS712实际输出
    float current = (actual_voltage - 2.5f) / 0.185f; // 5V供电: 零电流=2.5V
    return current;
}
```

## 三、CubeMX硬件配置参数

### 3.1 时钟树 (8MHz HSE → 250MHz SYSCLK)

```
HSE (8MHz)
  │
  ├─ PLLM = /2  ──→ 4MHz
  │    └─ PLLN = ×125 ──→ 500MHz VCO (Wide Range)
  │         ├─ PLLR = /2  ──→ 250MHz → SYSCLK, HCLK
  │         ├─ PLLP = /2  ──→ 250MHz → APB1/2/3 Timer Clocks
  │         └─ PLLQ = /4  ──→ 125MHz → ADC/I2C/USART Clocks
  │
  ├─ HSI (64MHz) → 备用
  └─ LSE (32.768kHz) → RTC + IWDG
```

| 总线 | 频率 | 分频 |
|------|------|------|
| SYSCLK | **250 MHz** | HSE → PLLR |
| HCLK | **250 MHz** | ÷1 |
| APB1 | **125 MHz** | ÷2 |
| APB2 | **125 MHz** | ÷2 |
| APB3 | **125 MHz** | ÷2 |
| TIM1/2/3... (APB2) | **250 MHz** | ×2 (当APB prescaler≠1时) |
| ADC | **125 MHz** | PLLQ÷2 |
| FLASH Latency | **5 WS** | 250MHz需要 |

### 3.2 外设时钟

| 外设 | 总线 | 时钟频率 | 备注 |
|------|------|---------|------|
| I2C1 | APB1 (125MHz) | 125MHz | Timing=0x10707DBC → 400kHz SCL |
| USART1 | APB2 (125MHz) | 125MHz | 921600bps, DIV=8.482 ≈ BRR=8.5 |
| USART3 | APB1 (125MHz) | 125MHz | 9600bps (PMS7003), BRR=813.8 |
| ADC1 | AHB (250MHz) | 125MHz (÷2) | 12-bit, 24.5 cycles sampling |
| TIM1 | APB2 (250MHz) | 250MHz | 20kHz PWM: PSC=0, ARR=12499 |

### 3.3 DMA配置

| DMA通道 | 外设 | 方向 | 优先级 | 用途 |
|---------|------|------|--------|------|
| GPDMA1_CH0 | USART1_RX | P→M | High | OpenMV 921600图像数据 |

| GPDMA1_CH2 | ADC1 | P→M | Medium | ACS712 100Hz连续采样 |

## 四、I2C总线拓扑

```
VCC3.3 ─── [4.7kΩ×2 外接上拉] ─── I2C1_SCL(PB6) / I2C1_SDA(PB7)
                                        │
                    ┌───────────────────┼───────────────────┐
                    │                   │                   │
                 SHT40              SGP40              SSD1306
               addr:0x44          addr:0x59          addr:0x3C
            (±1.8%RH/±0.2°C)   (VOC 0-500)      (128×64 OLED)
```

- 地址无冲突: 0x44 / 0x59 / 0x3C
- 速率: 400kHz (Fast Mode)
- 上拉: 需外接4.7kΩ×2 (板上I2C2有上拉但I2C1没有)
- PCB走线长度 < 10cm (I2C总线电容 < 400pF)

## 五、电机驱动接口 (DRV8833模式)

```
STM32H562                DRV8833模块           BLDC Motor
─────────              ────────────          ──────────
PA8 (TIM1_CH1) ────→   PWM/ENA      ────→   Speed
PC5 (GPIO)     ────→   DIR/PHASE    ────→   Direction
PD7 (GPIO)     ────→   nSLEEP       ────→   Enable
GND            ────→   GND          ────→   GND
                       VMOT(24V)    ────→   24V Power
                       MOTOR_A      ────→   Motor Phase A
                       MOTOR_B      ────→   Motor Phase B
```

- PWM频率: 20kHz (超出人耳听觉范围，无啸叫)
- 6档占空比: 0% / 30% / 45% / 60% / 80% / 100%
- 软启动: 斜坡速率10step/10ms，500ms到达全速
- 过流保护: 软件7.5A预警 + 8A紧急停止

## 六、ACS712硬件分压电路

```
ACS712模块 (5V供电)
    │
    VOUT (0A=2.5V)
    │
    ├──── [10kΩ] ────┬──── [10kΩ] ──── GND
    │                  │
    │             STM32 PC4 (ADC1_INP4)
    │             V_ADC = VOUT/2
    │             范围: 1.25V±(0.185V/A × I × 0.5)
    │             0A→1.25V, +5A→1.712V, -5A→0.788V
    │             (全范围在0-3.3V内 ✓)
```
